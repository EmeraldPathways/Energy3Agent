# Project Starter Template

Copy this folder into a new project root at the start of work.

## Includes

- `AGENTS.md`
- `PROJECT.md`
- `phases.md`
- `.agent-handoff/`
- `.ai-codex/`

## First setup steps

1. Fill in `PROJECT.md`.
2. Fill in `.ai-codex/index.md` and add only the docs you need.
3. Put the current task into `.agent-handoff/codex-task.md`.
4. Let Codex create the prompt.
5. Let Cline or DeepSeek implement it and report back in `.agent-handoff/cline-result.md`.
6. Record validation in `.agent-handoff/validation-log.md`.
